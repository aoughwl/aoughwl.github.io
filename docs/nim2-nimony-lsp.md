# nimony-lsp (Nim 2) — deprecated

The original **Nim 2** `nimony-lsp` — a Language Server + VSCode extension built
on Nimony's `idetools`. It is **complete and fully featured**, and **deprecated**:
superseded by **[aowllsp](aowllsp)**, the ground-up rewrite written *in* Nimony
(self-owned, and JS-compilable for the in-browser IDE).

[Archived repo → github.com/aoughwl/nim2-nimony-lsp](https://github.com/aoughwl/nim2-nimony-lsp)

Everything current lives at **[aoughwl.github.io](/)**.
