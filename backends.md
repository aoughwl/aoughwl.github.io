# Nimony Backends

Alternative code-generation targets for the nimony compiler — compile one nimony
program to the web, TypeScript, or Python. Each page below is the canonical docs
for its repo.

| Backend | Repo | Status |
|---|---|---|
| [nimony-web](docs/nimony-web) — JavaScript + WebAssembly backends & async runtime | `aoughwl/nimony-web` | private repo · public docs |
| [nimony-ts](docs/nimony-ts) — idiomatic TypeScript backend | `aoughwl/nimony-ts` | early scaffold · private |
| [nimony-py](docs/nimony-py) — idiomatic Python backend | `aoughwl/nimony-py` | early scaffold · private |
| [nimony-hl](docs/nimony-hl) — shared High-Level IR | `aoughwl/nimony-hl` | early scaffold · private |
